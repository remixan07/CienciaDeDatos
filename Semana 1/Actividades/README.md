¿Qué se hizo?

En esta carpeta se realizaron actividades introductorias, como la instalación de herramientas (Python, Visual Studio Code y Git), así como lecturas sobre programación y conceptos básicos. También se incluyeron ejercicios de comprensión relacionados con lenguajes de programación, paradigmas y nociones generales de desarrollo.

¿Para qué se hizo?

Se llevó a cabo con el objetivo de comprender los fundamentos de la programación, preparar el entorno de trabajo y desarrollar una base teórica que permita avanzar hacia la práctica.
