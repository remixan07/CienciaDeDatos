¿Qué se hizo?

Se desarrolló una actividad integradora en la que se aplicaron conceptos como:

Uso de variables
Entrada y salida de datos
Estructuras básicas de control
Resolución de problemas mediante algoritmos

Además, se utilizaron archivos en formato JSON para estructurar información o representar datos utilizados en la actividad.

¿Para qué se hizo?

El propósito fue aplicar de manera práctica los conocimientos adquiridos durante la semana, evaluar el nivel de comprensión y simular situaciones reales donde se requiere organizar y procesar información.
